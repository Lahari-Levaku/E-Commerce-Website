# E-Commerce-Website
This project features an interactive Power BI dashboard that analyzes e-commerce performance across key business metrics, including total revenue, total orders, website visits, add-to-cart actions, conversion rate, and cart abandonment rate. It also provides insights into revenue by category, top-performing products, and daily sales trends.

Skills demonstrated:
- Data cleaning and transformation
- DAX measure creation
- Data modelling
- Interactive dashboard design
- Turning raw data into actionable insights

This dashboard offers a clear, data-driven view of e-commerce performance and supports better decision-making for product, marketing, and sales teams.
